# Refactory Academy Website Clone

This repository contains the HTML and CSS files for a **clone of the Refactory Academy website**, created as part of a web development assignment. The project replicates the layout, sections, and styling of the Refactory Academy site for learning purposes.

## Features

- **Responsive Navbar** with clickable logo
- **Hero Section**: Refactory Academy introduction
- **Apprenticeship Section** with call-to-action
- **Featured Courses** with images and cards
- **10X Program Section** with partner logos linking to their websites
- **Footer**: Social media icons, resources, and contact info
- Fully **responsive design** for mobile and desktop
